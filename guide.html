예약 등록 — 4단계 위저드

✅
로그인 후 예약하기 탭을 선택하면 4단계 과정으로 입고 도크를 예약할 수 있습니다.
1

입고 차량의 종류와 작업 방식을 선택합니다. 이 조합에 따라 소요 시간(슬롯 수)이 달라집니다.

2

날짜 선택

2주간 달력에서 입고 날짜를 선택합니다. 날짜 셀의 색상으로 예약 가능 여부를 확인할 수 있습니다.

내 예약현황 (히스토리)

  • 내 예약현황 탭에서 최근 90일~향후 90일 예약 목록 조회 가능
  • 상태 필터 (전체 / 승인대기 / 확정 / 완료 / 취소) 로 분류 조회
  • 승인대기 상태의 예약만 취소 버튼 표시
  • CSV 다운로드 버튼으로 예약 목록 엑셀 저장 가능

예약관리 캘린더

  • 예약관리 탭에서 주간 캘린더로 내 예약과 타사 예약 현황 확인
  • 내 예약: 초록 으로 표시, 상세 정보 노출
  • 타사 예약: 회색 '예약완료' 로만 표시 (상세 숨김)
  • 이전주 / 다음주 네비게이션 가능

도크_예비 이용 조건

  • 기본적으로 비활성 상태 (예약 불가)
  • 관리자가 특정 날짜에 활성화한 경우에만 이용 가능
  • 물량이 많은 날, 특별 운영일에 활성화됨
  • 예약 화면에서 도크_예비 컬럼이 표시되면 예약 가능
FAQ

자주 묻는 질문

예약했는데 '승인대기' 상태입니다. 언제 확정되나요? ▼
고객사가 예약하면 자동으로 '승인대기' 상태로 생성됩니다. 물류센터 운영팀이 확인 후 '확정'으로 변경합니다. 긴급 확정이 필요한 경우 운영팀에 직접 연락 바랍니다.
확정된 예약을 취소하고 싶습니다. ▼
고객사 화면에서는 '승인대기' 상태의 예약만 직접 취소할 수 있습니다. '확정' 이후 취소는 물류센터 운영팀에 요청해야 합니다. 단, 당일(D+0)과 다음 영업일(오전 11시 이후)은 마감 규칙으로 취소도 불가합니다.
오전 11시 이후인데 내일 예약을 못 하고 있습니다. ▼
D-1 오전 11시 마감 규칙에 따라, 당일 11시 이후에는 다음 영업일 예약이 차단됩니다.
예외적으로 운영팀이 마감을 임시 해제할 수 있으니, 긴급한 경우 운영팀에 문의하세요.
슬롯 선택 화면에서 특정 도크가 보이지 않습니다. ▼
고객사 화면에는 해당 업체에 배정된 도크만 표시됩니다. 배정 도크 확인이 필요하면 운영팀에 문의하세요. 도크_예비는 관리자가 해당 날짜를 활성화한 경우에만 표시됩니다.
예약 한도 초과로 예약이 거부됩니다. ▼
업체별로 하루 최대 예약 건수(총 대수), 차량유형별, 작업유형별 한도가 설정되어 있습니다. 한도 조정이 필요한 경우 운영팀에 문의하세요. 슬롯 선택 화면 상단에서 현재 사용량과 잔여 한도를 확인할 수 있습니다.
발주번호를 모르는데 예약할 수 있나요? ▼
발주번호는 필수 입력 항목입니다. 발주번호가 없는 상태로는 예약이 불가합니다. 발주번호 확인 후 예약을 진행해 주세요.
12시에 예약하려고 하는데 선택이 안 됩니다. ▼
12:00 ~ 13:00는 점심시간으로 자동 차단됩니다. 해당 시간을 포함하는 슬롯은 예약이 불가합니다. 12시 이전 또는 13시 이후 시간을 선택해 주세요.
로그인이 안 됩니다. ▼
아래 사항을 확인해 주세요:
1. 업체코드와 비밀번호를 정확히 입력했는지 확인 (대소문자 구분)
2. 계정이 비활성 상태인 경우 로그인이 차단됩니다 → 운영팀에 활성화 요청
3. 비밀번호를 잊어버린 경우 운영팀에 초기화 요청
